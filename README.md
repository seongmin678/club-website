## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Deployment:** GitHub Pages

---

## ✨ Key Features

### 1. Responsive Design
- 다양한 기기(모바일, 태블릿, PC)에서 최적화된 화면을 보여주는 반응형 웹 디자인을 적용했습니다.

### 2. Interactive UI
- CSS Animation과 JavaScript Event Listener를 활용하여 사용자 경험(UX)을 높이는 동적인 효과를 구현했습니다.

### 3. Club Information Management
- 동아리 소개, 프로젝트 아카이브, 멤버 목록 등을 한눈에 볼 수 있도록 직관적인 레이아웃으로 구성했습니다.

---

## 📂 Project Structure
- `index.html`: 웹사이트의 메인 구조 및 콘텐츠
- `styles.css`: 레이아웃 디자인 및 애니메이션 스타일링
- `script.js`: 동적 인터랙션 및 이벤트 처리 로직
- `images/`: 웹사이트에 사용된 이미지 리소스

---

## 💡 개발 포인트 및 트러블슈팅
- **웹 호스팅 최적화**: GitHub Pages를 활용하여 별도의 서버 비용 없이 정적 웹사이트를 성공적으로 배포했습니다.
- **이미지 최적화**: 웹 로딩 속도를 높이기 위해 이미지 용량 최적화 및 적절한 포맷 선택을 고민했습니다.
